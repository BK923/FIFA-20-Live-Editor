### FIFA 20 Live Editor
FIFA Live Editor (previously known as Career Mode Cheat Table) is a tool created in [cheat engine](https://github.com/cheat-engine/cheat-engine) which allows you to edit players, managers, transfer budget, scouts, job offers and many other things in your FIFA Career Mode.
